## Descrição do Projeto

# O projeto consiste em um sistema web para uma ONG de animais, onde os usuários poderão cadastrar animais para adoção, e os visitantes poderão visualizar os animais cadastrados e entrar em contato com a ONG para adotar um animal. (parcialmente implementado) (será integrado a um banco de dados no proximo termo.) 

# No site os usuários poderão efetuar cadastro de voluntários, animais e usuários, além de poderem visualizar os animais cadastrados e entrar em contato com a ONG para adotar um animal. (em construção)

# Ná pagina index.html o usuário verá um carrocel de imagens com alguns dos animais cadastrados.
# erá uma prévia da loja da ong, onde estará disponível 3 produtos para venda.(em construção).
# Logo abaixo tres cards com informações sobre a ong, e  botão para o adocão, doação e se voluntariar, onde sera redirecionado a sua respectiva pagina.

# Na pagina de adoção, o usuário poderá visualizar os animais cadastrados, e entrar em contato com a ONG para adotar um animal. Essa pagina será integrada ao banco de dados onde os animais cadastrados serão exibidos. (em construção).

# NO sistema SGEA o usuário poderá efetuar o cadastro de novos voluntários, animais, projetos (em construção). Por enquanto foi somente inserido na pagina inicial um formulário para cadastro de voluntários e uma tabela para visualização dos voluntários cadastrados. Onde atráves de um check-box o usuário poderá selecionar o voluntário que deseja excluir. Foi implementado utilizando vetores estaticos simulando um banco de dados, conforme solicitado pelo professor. Posteriormente com  a integração ao banco de dados, será movido para a sua respectiva sessão.


## Foi utilizando o framework Bootstrap 5 para o desenvolvimento do site, e Javascript para validação de formulários e algumas mascaras para formatação de campos.
